## 逐课知识点详细分析

### 第1课：排列组合1
- **文件**: 1-排列组合1.pptx (113页)
- **类型**: 编程/理论
- **知识点数**: 6个
- **GESP 8级**: 加法原理(8.1)、乘法原理(8.2)、排列(8.3)
- **详细知识点**:
  1. 加法原理（分类计数）- 完成一件事有n类办法，总方法数为各类方法数之和
  2. 乘法原理（分步计数）- 完成一件事需要n个步骤，总方法数为各步方法数之积
  3. 排列的概念 - 从n个不同元素中取出m个元素按顺序排列
  4. 排列数公式 P(n,m) = n!/(n-m)!
  5. 全排列 - n个元素的全排列数为n!
  6. 特殊排列问题 - 相邻问题、不相邻问题、定序问题

### 第2课：排列组合2
- **文件**: 2-排列组合2.pptx (69页)
- **类型**: 编程/理论
- **知识点数**: 6个
- **GESP 8级**: 排列(8.3)、组合(8.4)、杨辉三角(8.5)
- **详细知识点**:
  1. 组合的概念 - 从n个不同元素中取出m个元素组成一组
  2. 组合数公式 C(n,m) = n!/(m!(n-m)!)
  3. 组合数性质 - C(n,m)=C(n,n-m), C(n,m)=C(n-1,m-1)+C(n-1,m)
  4. 杨辉三角 - 组合数的几何表示
  5. 二项式定理 - (a+b)^n = ΣC(n,k)a^(n-k)b^k
  6. 排列组合综合应用

### 第3课：质数筛法

- **类型**: 编程
- **知识点数**: 4个
- **GESP映射**: 无直接映射
- **详细知识点**:
  1. 质数判定基础 - 试除法判断质数，O(√n)
  2. 埃拉托斯特尼筛法 - 筛除质数的倍数，O(nloglogn)
  3. 欧拉筛法（线性筛）- 每个合数只被最小质因子筛除，O(n)
  4. 质数筛法的应用 - 区间质数、质数计数、质因数分解
